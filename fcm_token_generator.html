Click the button below to generate your device token.

Before You Start

Both fcm_token_generator.html and firebase-messaging-sw.js must have your real Firebase config values filled in before use. Replace every YOUR_* placeholder in both files with values from Firebase Console → Project Settings → Your Apps → Web App.

This page must be served over HTTPS — service workers do not work on file:// URLs. The easiest free option is GitHub Pages:

  1. Push both files to your alpha-scout repo.
  2. Go to repo → Settings → Pages.
  3. Under Source, select Deploy from a branch.
  4. Set branch to main and folder to / (root), then click Save.
  5. After a minute, open https://<your-username>.github.io/alpha-scout/fcm_token_generator.html.

Generate Your Token

  1. Click Generate FCM Token above.
  2. Allow notification permissions when prompted by your browser.
  3. Copy the token that appears.
  4. Go to your GitHub repo → Settings → Secrets and variables → Actions.
  5. Add the token as FCM_DEVICE_TOKEN_1 (Mac) or FCM_DEVICE_TOKEN_2 (Android).
  6. Repeat on each device you want to receive alerts on.
